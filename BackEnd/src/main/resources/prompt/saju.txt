다음 사용자의 사주 정보를 바탕으로 사주 명리학적 분석 결과를 한국어 JSON 형식으로 정리해 주세요.

입력 정보:
- 이름: {{name}}
- 생년월일: {{birthDate}} (예: 1993-07-21)
- 생시: {{birthTime}} (예: 오전 9시, 15:00 등)
- 성별: {{sex}}

요구사항:
1. 분석 결과는 반드시 아래의 **출력 형식(format)**에 **정확히 일치하는 JSON 구조로 출력**해 주세요.
2. `"summary"`: 사주의 전체적인 흐름과 해석을 2~3문장으로 요약해 주세요.
3. `"ohaeng"`: 오행의 구성과 균형(금·목·수·화·토의 분포와 과부족)을 설명해 주세요.
4. `"personality"`: 기질적 성격, 대인관계, 성향 등을 분석해 주세요.
5. `"luck"`: 초년·중년·말년 운의 흐름, 재물운, 연애운 등을 간단히 서술해 주세요.
6. `"suggestion"`: 전반적인 조언이나 유의할 점을 자연스럽게 작성해 주세요.
7. 응답은 반드시 **JSON만 포함**해야 하며, 설명, 예시, 인삿말은 포함하지 마세요.
8. 모든 내용은 **전문가 수준의 진지한 문체**로, **한국어**로 작성해 주세요.

출력 형식:
{{format}}
