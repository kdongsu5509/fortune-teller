다음 이미지 URL을 바탕으로 얼굴 인상에 대한 분석을 관상 전문가의 시선으로 진행해 주세요.

- 이미지 URL: {{imageUrl}}

요구 사항:
1. 분석 결과는 반드시 아래의 **출력 형식(format)**에 **정확히 일치하는 JSON 구조로 출력**해 주세요.
2. 출력 형식은 엄격한 출력 사양이며, **구조나 키 이름을 변경하거나 생략하지 마세요**.
3. 모든 필드(`summary`, `keywords`, `features`, `advice`)는 **반드시 포함되어야 하며**, `null` 없이 **구체적인 값**으로 작성해 주세요.
4. `features`는 다음 4개의 항목을 포함해야 합니다:
   - `eyes`
   - `nose`
   - `mouth`
   - `forehead`
   각 항목은 아래 2개의 필드를 가져야 합니다:
   - `description`: 외형적 특징 (예: 크기, 형태, 인상 등)
   - `meaning`: 해당 부위에 대한 **관상학적 해석** (성격, 대인관계, 운세적 의미 등)
5. `summary`: 얼굴 인상의 전반적인 해석을 2~3문장으로 요약해 주세요.
6. `keywords`: 분석을 대표하는 핵심 단어 3~5개를 배열로 포함해 주세요.
7. `advice`: 인상과 관련된 전반적인 조언이나 주의점을 진지하게 작성해 주세요.
8. 응답은 반드시 **JSON만 포함**해야 하며, 인삿말, 예시 문구, 설명 등은 절대 포함하지 마세요.
9. 문체는 **관찰 기반 전문 분석 문체**로 작성하고, `~한 인상을 줍니다`와 같이 **존댓말**을 사용해 주세요.
10. 출력 언어는 **한국어**여야 합니다.

출력 형식:
{{format}}
